Getting Started

Introduction

Matrix.css is a comprehensive CSS framework inspired by The Matrix aesthetic. It provides a dark theme with neon green accents, a complete component library, and utility classes for building futuristic, cyberpunk-inspired interfaces.

The framework includes everything from basic UI components to advanced Matrix-themed effects, making it perfect for cybersecurity dashboards, sci-fi themed websites, or any project that wants to capture that distinctive Matrix feel.

Note: Matrix.css is designed as a standalone CSS framework, with no JavaScript required for most components. However, some interactive components may benefit from JavaScript for full functionality.

Installation

You can quickly include Matrix.css in your project using one of the following methods:

Direct Download

<link rel="stylesheet" href="path/to/matrix.css">

CDN

<link rel="stylesheet" href="https://cdn.example.com/matrix-css/1.0/matrix.css">

NPM

npm install matrix-css

Then import it in your JavaScript:

import 'matrix-css/dist/matrix.css';

Layout

Containers

Use container elements to center and pad your content. The Matrix.css container adapts to screen size for responsive design.

Grid System

The grid system is a flexbox-based layout that lets you define columns which automatically wrap on smaller screens.

Spacing

Utility classes for margin and padding (e.g., .m-2, .p-3) help space elements consistently in a cyberpunk style.

Alerts

Alerts display important messages with contrasting colors (e.g., red for errors, yellow for warnings) and can include flicker or blink effects for emphasis.

Cards

Cards have a holographic look with semi-transparent backgrounds and a glowing border for a futuristic aesthetic, often containing a header, body, and optional footer.

Badges

Badges are small count or label elements, styled as bright pill shapes to draw attention (for example, notification counts or status labels).

Progress Bars

Progress bars animate to show completion progress in a retro terminal style, usually with a neon green progress fill and a dark background track.

Spinners

Spinners provide a loading indicator inspired by classic command-line cursors or rotating ASCII glyphs, indicating that a process is ongoing.

Advanced Components

Neural Network

Visualizes a neural network with a grid of nodes and animated connections, fitting the theme's tech aesthetic.

Terminal

The Terminal component emulates a terminal window, including a blinking cursor and typewriter effect for text output.

Glitch Text

Glitch Text uses CSS animations to intermittently distort and scramble text, mimicking a hacked or glitchy display.

Identity Card

An Identity Card shows a user profile in a futuristic ID format, complete with flicker and scanline effects.

Code Rain

The Code Rain component produces the signature falling code animation from The Matrix, often used as a full-screen background effect.

Utilities

Visual Effects

Includes CSS utilities for adding visual effects like flicker, glitch, and glow to elements, enhancing the cyberpunk ambiance without extra scripts.

Animations

Provides pre-defined CSS keyframe animations (e.g., pulse, fade, bounce) to easily animate elements in a futuristic style.

Transforms

Offers utility classes to apply CSS transforms (rotate, scale, skew) with Matrix-themed transitions, allowing dynamic element manipulation.

Responsive Design

Matrix.css is built to be fully responsive, with grid and utility classes that adapt layouts to different screen sizes for an optimal experience on all devices.

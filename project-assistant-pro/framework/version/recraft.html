Introduction

Matrix.css is a comprehensive CSS framework inspired by The Matrix aesthetic. It provides a dark theme with neon green accents, a complete component library, and utility classes for building futuristic, cyberpunk-inspired interfaces.

The framework includes everything from basic UI components to advanced Matrix-themed effects, making it perfect for cybersecurity dashboards, sci-fi themed websites, or any project that wants to capture that distinctive Matrix feel.

Note: Matrix.css is designed as a standalone CSS framework — no JavaScript required for most components. However, some interactive components may benefit from JavaScript for full functionality.

Installation

You can quickly include Matrix.css in your project using one of the following methods:

Direct Download

<link rel="stylesheet" href="path/to/matrix.css">

CDN

<link rel="stylesheet" href="https://cdn.example.com/matrix-css/1.0/matrix.css">

NPM

npm install matrix-css

Then import it in your project:

import 'matrix-css/dist/matrix.css';

Typography

Matrix.css uses two primary fonts: a monospace font for the code-like aesthetic and a secondary retro-styled font for accents. By default, the framework uses Courier New (monospace) as the main font and VT323 as the alternate retro font.

Font sizes are set on a consistent scale to establish clear hierarchy, and line heights are optimized for legibility in text-dense layouts.

Base Font

The base font (monospace) ensures all text has a uniform, terminal-like appearance. You can override the --m-font-main CSS variable to use a different font if desired.

Alternate Font

The alternate font, used for special elements like the logo or titles, is a stylized retro typeface. The default VT323 gives an authentic old-school computer look and feel.

Containers

Containers are used to limit the width of content and center it. By default, .m-container has a max-width of 1200px and horizontal padding. Wrap your page content in a container for consistent spacing on large screens.

<div class="m-container"> ... </div>

Grid System

Matrix.css includes a flexbox-based 12-column grid. Use .m-row for rows and .m-col-* classes for columns (e.g., .m-col-6 for half-width).

Spacing

The framework provides utility classes for spacing (margin and padding). For example, .m-mb-4 applies a medium bottom margin, and .m-py-5 applies large vertical padding.

Buttons

Matrix.css offers various button styles and sizes. All buttons use the base .m-btn class, with modifiers like .m-btn-primary, .m-btn-lg, etc., for different appearances.

Alerts

Use alert classes such as .m-alert with modifiers like .m-alert-info, .m-alert-danger, etc., to create notification banners.

Cards

Cards are flexible content containers. Use .m-card for the card, with .m-card-body, .m-card-title, and .m-card-text for content within.

Card Title

Badges

Badges are small inline elements that display counts or status labels. Use the base .m-badge class with modifiers like .m-badge-primary for different colors.

Progress Bars

Create progress bars with .m-progress (container) and .m-progress-bar (inner bar). Adjust the inner width style for the progress percentage.

Spinners

Spinners indicate loading states. Use .m-spinner for a default spinner and .m-spinner-sm for a smaller version.

Neural Network

The Neural Network component visualizes a network of nodes and connections, evoking the feel of a digital neural net.

Terminal

The Terminal component provides a stylized command-line interface window.

user@matrix:~

Glitch Text

Add the m-glitch class to text to give it a glitchy, digital distortion effect. Use the data-text attribute on the element to hold the uncorrupted text.

Identity Card

The Identity Card is a profile card with a Matrix theme, complete with code rain background and glitch effects.

Code Rain

The iconic Matrix "code rain" effect can be applied by adding the m-code-rain class to any container. This will animate a cascading rain of green symbols within that element.

Visual Effects

Matrix.css provides special effect classes like m-glow (neon glow), m-scanline (moving scanline overlay), and m-holo (holographic shimmer) to enhance the cyberpunk vibe.

Animations

A variety of CSS animations are available via classes (e.g., m-fade-in, m-bounce, m-pulse) to bring components to life.

Transforms

Utility classes for CSS transforms (like m-rotate-45, m-scale-110) allow you to easily rotate, scale, and translate elements without writing custom CSS.

Responsive Design

Matrix.css is mobile-first and includes classes to hide or show content at certain breakpoints (e.g., m-hide-sm, m-show-lg). The framework’s use of relative units ensures it adapts seamlessly to different screen sizes.

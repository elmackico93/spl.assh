Overview

Matrix.css is a comprehensive CSS framework inspired by The Matrix aesthetic. It provides a dark theme with neon green accents, a complete component library, and utility classes for building futuristic, cyberpunk-inspired interfaces.

The framework includes everything from basic UI elements to advanced Matrix-themed effects, making it perfect for cybersecurity dashboards, sci-fi websites, or any project that wants to capture that distinctive Matrix feel.

Note: Matrix.css is designed as a standalone CSS framework (no JavaScript required for core components). However, some interactive examples (like the theme switcher and terminal effects) use JavaScript for enhanced functionality.

Installation

You can quickly include Matrix.css in your project using one of the following methods:

Direct Download

<link rel="stylesheet" href="path/to/matrix.css">

CDN

<link rel="stylesheet" href="https://cdn.example.com/matrix-css/2.0/matrix.css">

NPM

npm install matrix-css

Then import it in your JavaScript:

import 'matrix-css/dist/matrix.css';

Grid System

The Matrix.css grid system is a 12-column responsive flexbox grid. Use a m-row container for rows and columns with classes like m-col-md-6 for 6/12 width on medium screens. Columns will automatically wrap on smaller devices. You can also offset columns (e.g., m-offset-md-3 for a 25% left offset on medium screens).

Utilities

Matrix.css includes many utility classes for quick styling without custom CSS:

  • .m-mt-*, .m-mb-*, .m-pt-*, etc. – margin and padding utilities (e.g., m-mt-4 adds a top margin).
  • .m-text-center, .m-text-uppercase – text alignment and transformation utilities.
  • .m-flex, .m-flex-center, .m-flex-between – flexbox container helpers for layout.
  • .m-border, .m-divider – border and divider utilities to separate content.

These utilities (and more) follow a consistent naming scheme (prefixed with m-) to enable rapid development of layouts directly in HTML.
